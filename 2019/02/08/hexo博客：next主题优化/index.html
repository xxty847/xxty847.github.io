动态线条 canvas_nest

$ git clone https://github.com/theme-next/theme-next-canvas-nest source/lib/canvas-nest

动态彩带 canvas_ribbon

  • 主题目录下运行以下命令
$ git clone https://github.com/theme-next/theme-next-canvas-ribbon source/lib/canvas-ribbon
