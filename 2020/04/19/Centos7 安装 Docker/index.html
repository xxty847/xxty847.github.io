Centos7 安装 Docker 和 Docker Compose

发表于 2020-04-19 分类于 Centos Valine：
  • Docker：https://www.docker.com/
  • 官方文档：https://docs.docker.com/engine/install/centos/
  • Docker Compose：https://github.com/docker/compose/releases/

安装 Docker

(1) 使用yum命令进行安装Docker

$ yum install -y docker
$ yum list installed | grep docker    # 查看是否安装成功

(2) 启动docker服务

$ systemctl start docker
$ systemctl status docker    # 查看是否启动成功

（3）查看docker已经安装的镜像

$ docker images        # 查看docker已经安装的镜像,裸机状态下为空
$ docker version        # 查看docker版本

手动安装 Docker-ce

（1）升级yum

$ yum update

（2）删除旧 docker
  旧版本的 docker 的叫做 docker 或者 docker-engine，如果系统中已经安装旧版本，则需要删除。通过以下命令删除旧的 docker 版本。

$ yum remove docker docker-common docker-selinux docker-engine

(3) 安装最新版Docker
  新的 docker 叫做 docker-ce ，如果第一次安装 docker-ce 需要设置 docker-ce 的 yum 源。用下面命令来增加 docker-ce 的 yum 源。

安装依赖
$ sudo yum install -y yum-utils device-mapper-persistent-data lvm2    

配置仓库（建议使用下一步阿里云加速镜像）    
$ sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
配置仓库(阿里云加速镜像)
$ sudo yum-config-manager --add-repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

排序查看仓库内文件
$ yum list docker-ce --showduplicates | sort -r

安装最新版Docker
$ sudo yum install docker-ce docker-ce-cli containerd.io

（4）配置启动Docker

启动Docker    
$ sudo systemctl start docker

查看Docker版本    
$ sudo docker version

设置Docker开机自动启动    
$ sudo systemctl enable docker

Docker官方一键安装脚本

下载一键安装脚本

$ curl -fsSL https://get.docker.com -o get-docker.sh

运行一键安装脚本(阿里云加速镜像)

$ sudo sh get-docker.sh --mirror Aliyun

安装 Docker Compose

方法一：命令安装

官网选择版本

以下命令手动修改版本号,例如1.25.4
$ curl -L https://github.com/docker/compose/releases/download/1.25.4/docker-compose-`uname -s`-`uname -m` -o /usr/local/bin/docker-compose

添加执行权限
$ chmod +x /usr/local/bin/docker-compose

检查Docker Compose版本
$ docker-compose version

方法二：下载安装包

官网选择版本，下载docker-compose-Linux-x86_64到本地或者服务器中。

xftp传输文件失败的解决方法：

$ chmod -R 777 /usr/local/bin/    # 添加权限

更名为docker-compose,并移动到 /usr/local/bin 目录下

添加执行权限
$ chmod +x /usr/local/bin/docker-compose

检查Docker Compose版本
$ docker-compose version
